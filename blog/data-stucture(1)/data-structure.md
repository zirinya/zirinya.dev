Linear data structures

1. Arrays
   อาเรย์ คือ ประเภทของข้อมูลที่สามารถเก็บข้อมูลประเภทเดียวกันแบบเป็นลำดับได้ โดยข้อมูลนั้นจะอยู่ในชุดเดียวกัน
   ทำให้การจัดการข้อมูลง่ายขึ้นแทนที่เราจะกำนดค่าเยอะๆ ก็ยุบรวมกันไว้ในตัวแปลเดียวเราเรียกค่าที่อยู่ในชุดข้อมูลว่า element และถ้าเกิดกำหนดตัวเแปรหลายๆตัวตัวแปลเหล่านั้นจำมีชื่อไม่เหมือนกัน แตาถ้าเป็น array เราสามารถเข้าถึงข้อมูลได้ด้วย index ซึ่งเป็นค่าอ้างที่ไม่ซ้ำกัน(uniquely )
2. Stack
   สแตกคือการเก็บข้อมูลแบบเรียงต่อกันและบนำดับการเก็บข้อมูล
   การเรีรยงอันดับอาจจะเป็น last in fist out LIIFO
   เหมือนการกินมันฝรั่งแบบบในกระบอกคืออันที่เข้ามาหลังสุดออกไปก่อน
   //img
3. Queue
   สแตกคือการเก็บข้อมูลแบบเรียงต่อกันและบนำดับการเก็บข้อมูล
   การเรียงอันดับเหมือนสแตกแต่คิวจะเหมือนการเข้าคิวซื้อขานม FIFO (First in First out)
4. Linked List
   โครงงสร้างของการเก็บข้อมูลของลิงคค์ลิสต์คือจินตนาการว่า
   ข้อมูลของเราเป็น node ภายใน node นั้นมีข้อมูลอยู่ข้างใน
   เราสามารถให้ node นั้นชีวี้ไปอิกnode ได้

   ดูแบบนี้เหมือน link list จะเรียงต่อกันเหมือนอาเรย์
   แล้วมันต่างกันยังไงหรอ
   ตัวอย่างบสขมมติว่าเราจะinsert element ใน index ที่ 3 เราต้อง shift ตำแหน่งที่ 4 เข้าไปซึงในส่วของ linklist นั้นสามารถกำหนดไปได้เลยแต่แค่เปลี่ยนตัวที่ชี้ไปเอง ซึ่งในส่วนขอ time complexity นั้น จะเป็น O(1) ส่วน array นั้นจะเผ็น O(n)

Non linear data structures

5. Graph Data Structure
   ประกอบด้วย nodes มี egde ในกาีเชื่อโยงเข้าหากัน
   ตัวอย่างการเก็บข้อมูลเช่นอย่าง array 2 d

6. Trees Data Structure
   โครงสร้างของข้อมูลแบบต้นไม้คือเป็นข้อมูลที่จัดเรียงเป็นลำดับชั้น(hierarchal ) ประกอบด้วย central node, structural nodes, sub nodes เชื่อมต่อกันด้วย (edges)
   เราสามารถเปรียบเทียบกับต้นไม้ได้ตามชื่อคือมีรากมีกิ้งก้านเชื้อมถึงกัน
